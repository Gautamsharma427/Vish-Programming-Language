## Project Structure:
- lexer.py contains the lexer (version 1). Future lexers might be added like this also.
- main.py combines everything made until now and tries to run everything.
- parser.py will contain the parser(not yet ready)
- Docs folder contains the documents related to the working of the project:
    - Discussing working of individual files and their roadmap and logic flow(lexer, parser etc.)
    - other Markdown files which will be required by the project or improve the readability of the project in anyway.
- .gitignore includes all the files which aren't pushed to github.
- main.vish provides a overall preview of all the types of blocks which should be available in the final version of Vish programming language
- addition.vish is a program to perform addition b/w two numbers written according to the syntax of vish programming language.
- the folder "learning" are random files and folders i created for the purpose of learning something while making this project.
